VS Code 主題外掛 Material Theme 存在惡意行為，已下架。安裝量超過 900 萬

VS Code 業界消息 2025/02/27 goddaneel 0

來自川陀大學圖書館【開源雜談】第6期・緊急消息專欄的消息：兩款安裝量超過 900萬的知名 VS Code 外掛存在惡意行為，已從應用市場下架，提醒已安裝使用者解除安裝：

  • ｛Material Theme｝適用於 VSCode 的開源 Material 風格主題
  • ｛Material Theme Icons｝適用於 VSCode 的開源 Material 風格圖示主題

goddaneel 還說道：

  1. 請立即停止使用上述由 Equinusocio 開發維護的這兩款 VSCode 外掛，這些外掛目前被報告具有惡意行為，且 GitHub 提交記錄及開放原始碼疑似被清空。
  2. 相關外掛目前已從 Visual Studio Marketplace 下架並封禁開發者，但 Open VSX 似乎尚未進行處理。
  3. 相關外掛的危險版本似乎進行了防解除安裝處理，常規解除安裝會進入循環。如不幸使用了危險版本，你可能需要從外部手動解除安裝。
  4. 相關問題的原始討論目前已被刪除，如有需要可查閱存檔［1；2］。相關外掛的解除安裝方法及後續消息跟進，可參考 Marketplace 的 GitHub issue 和 Hacker News。

如果你安裝了這兩個外掛，務必解除安裝，在這裡繼續討論：https://meta.appinn.net/t/topic/66978
