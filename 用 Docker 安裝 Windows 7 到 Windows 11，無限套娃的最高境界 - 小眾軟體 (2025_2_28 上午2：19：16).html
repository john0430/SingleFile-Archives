用 Docker 安裝 Windows 7 到 Windows 11，無限套娃的最高境界

線上應用 Windows 2024/01/26 青小蛙 0

dockur/windows 是一個可以在 Docker 中運行從 Windows 7 到 Windows 10、11 系統的開放原始碼專案，然後通過瀏覽器訪問這台套娃 Windows 系統。需要大記憶體、KVM 支援。@Appinn

萬物皆可 Docker

三大作業系統：Windows、macOS、Linux。用 Docker 運行 Linux 是它的老本行，大多數 Docker 鏡像都基於 Linux。

在 Docker 運行 macOS 也不是不行，比如：https://meta.appinn.net/t/topic/47790

現在，在 Docker 運行 Windows 也來了。

dockur/windows

dockur/windows 項目運行起來非常簡單，一句話：

1
docker run -it --rm -p 8006:8006 --device=/dev/kvm --cap-add NET_ADMIN --stop-timeout 120 dockurr/windows

默認情況下，會安裝 Windows 11，並且會分配 2 個 CPU 核心和 4 GB 記憶體，以及 64GB 硬碟大小。

如果你要安裝其他系統，只需要在命令中加入 -e win10 即可：

  • win11 = Windows 11
  • win10 = Windows 10
  • win81 = Windows 8.1
  • win22 = Windows Server 2022
  • win19 = Windows Server 2019
  • win16 = Windows Server 2016
  • tiny11 = Tiny 11 (下載很慢)
  • win7 = Windows 7 (下載也很慢)

上述系統，下載很慢的來自 Archive 網站，其他的從微軟官網下載鏡像。

或者，你可使用自己的系統鏡像：

-e VERSION: "https://example.com/win.iso"

也可以使用本地鏡像：重新命名 custom.iso 並將其放入空的 /storage 資料夾中以跳過下載。

使用

安裝之後，就可以在瀏覽器打開 IP:8006 進行使用了
