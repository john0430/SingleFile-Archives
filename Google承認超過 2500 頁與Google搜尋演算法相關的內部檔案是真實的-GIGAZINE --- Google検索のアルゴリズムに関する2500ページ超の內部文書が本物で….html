Google検索のアルゴリズムに関する2500ページ超の內部文書が本物であることをGoogleが認める
Google承認，超過 2,500 頁關於 Google 搜尋演算法的內部檔是真實的


Googleの検索ランキングアルゴリズムに関する2500ページ超の內部文書「Google API Content Warehouse」がリークされました。この內部文書についてコメントを拒否していたGoogleですが、ついにこれが本物であることを認めました。
一份超過 2,500 頁的內部文檔“Google API Content Warehouse”關於 Google 的搜尋排名演算法已經洩露。 拒絕對內部檔案發表評論的Google終於承認它是真實的。

Google confirms the leaked Search documents are real - The Verge
Google確認洩露的 Search 檔案是真實的 - The Verge
https://www.theverge.com/2024/5/29/24167407/google-search-algorithm-documents-leak-confirmation


世界最大級の検索エンジンであるGoogle検索を運用するGoogleの、検索アルゴリズムに関する2500ページ超の內部文書「Google API Content Warehouse」がリークされました。この內部文書により、Googleがユーザーの情報やChromeのデータをどのように利用しているかの実態が明らかになっていたのですが、Googleはこの內部文書についてコメントを拒否していました。なお、Google API Content Warehouseがリークされることとなった経緯や、內部文書のリークにより明らかになった新情報などについては、以下の記事にまとめてあります。
運營著世界上最大的搜尋引擎之一 Google Search 的 Google 洩露了一份超過 2500 頁的關於搜尋演算法“Google API Content Warehouse”的檔。 這份內部檔揭示了Google如何使用使用者資訊和 Chrome 數據的實際狀態，但Google拒絕對這份內部檔案發表評論。 此外，以下文章還總結了Google API Content Warehouse洩露的背景和內部檔案洩露所透露的新資訊。

Googleの検索アルゴリズムに関する內部文書が流出、Chromeのデータをページランク付けに利用するなどGoogleのウソが明らかに - GIGAZINE
Google洩露有關Google搜尋演算法的內部檔、使用 Chrome 數據進行頁面排名等謊言被揭穿-GIGAZINE


Google API Content Warehouseによると、Googleはクリック數やChromeのユーザーデータ、Google検索でのウェブページのランキングには寄與しないと同社が言及したデータなどを、収集・使用している可能性が示唆されています。元々Google API Content Warehouseは、Googleの従業員が情報リポジトリとして利用していたものと推測されていますが、具體的にどのデータが検索ランキングアルゴリズムに利用されているかについては、明確になっていません。
根據Google API Content Warehouse 的說法，有人認為Google可能正在收集和使用點擊次數、Chrome用戶數據以及該公司提到的數據，這些數據對網頁在Google搜尋中的排名沒有貢獻。 據推測，Google API Content Warehouse 最初是被Google員工用作資訊倉庫的，但目前尚不清楚搜尋排名演算法使用了哪些具體數據。

Google API Content Warehouseについて、テクノロジーメディアのThe Vergeは記載されている情報が古くなっている可能性や、トレーニング目的でのみ使用されている可能性、収集されているものの検索関連では利用されていない可能性などを挙げながら、「ウェブを形成する最も重要なシステムのひとつであるGoogle検索の內部を、前例のないレベルで垣間見ることができる資料。ただし、依然として不明瞭な部分も多い」と表現しています。
關於Google API Content Warehouse，科技媒體 The Verge 指出，其中包含的資訊可能已過時，僅用於培訓目的，並且被收集但不用於搜尋上下文。 這份檔以前所未有的方式展示了Google搜尋的內部運作，Google搜尋是塑造網路的最重要系統之一。 但是，仍有許多不清楚的部分。


Google API Content WarehouseについてThe VergeがGoogleにコメントを求めたところ、同社の広報擔當者であるデイビス・トンプソン氏から「文脈から外れた、古い、または不完全な情報に基づいて検索について不正確な憶測をしないように注意を促します」「検索の仕組みやシステムが重視する要素の種類について、我々は広範囲にわたる情報を共有しており、同時に検索結果の完全性を操作から守るためにも取り組んできました」という回答が返ってきたそうです。
當 The Verge 聯繫 Google 就 Google API Content Warehouse 發表評論時，該公司發言人大衛斯·湯普森 （Davis Thompson） 表示，“我們敦促您不要根據斷章取義、過時或不完整的資訊對搜尋做出不準確的假設”，並且“我們想談談搜尋的工作原理以及系統強調的元素類型。共用了廣泛的資訊，同時我們一直在努力保護搜尋結果的完整性免受縱。

Googleのトンプソン氏はリークされた內部文書が本物であることを認めたものの、「文脈から外れた、古い、または不完全な情報」と表現しています。しかし、Google API Content Warehouseを公開したウェブサイト解析ツールを開発・提供しているSparkToroのランド・フィッシュキン氏と、SEO対策を行うiPullRankの創業者であるマイク・キング氏は、一部に古い情報が含まれるものの、ほとんどは2024年3月時點の最新情報であると確認しました。
Google的湯普森承認洩露的內部檔案是真實的，但將其描述為「斷章取義、過時或不完整的資訊」。 然而，SparkToro 的 Rand Fishkin，它開發並提供了一個網站分析工具，它已經發佈了 Google API Content Warehouse，以及 iPullRank 的創始人 Mike King，它做 SEO，雖然有些資訊已經過時了， 我們確認其中大多數是 2024 年 3 月的最新資訊。

なお、Google API Content WarehouseはSEO分野やマーケティング、パブリッシング業界全體に波紋を呼ぶ可能性が高いとThe Vergeは指摘。Googleはこれまで検索ランキングアルゴリズムの詳細を明かしていなかったものの、ビジネスでインターネットを利用する小規模の獨立系メディアからオンラインストアに至るまで、あらゆるユーザーにとって有益な「インターネットを支配するGoogleが何を考えているのか」に関する洞察を得るのにGoogle API Content Warehouseが役立つと、The Vergeは言及しました。
The Verge 指出，Google API Content Warehouse 可能會在整個 SEO 領域、行銷和發佈行業引起漣漪。 儘管 Google 之前尚未透露其搜尋排名演算法的細節，但 Google API 內容有助於深入瞭解“控制互聯網的 Google”對所有使用者都有好處，從小型企業媒體到使用 Internet 開展業務的在線商店。 The Verge 提到，Warehouse 有説明。
