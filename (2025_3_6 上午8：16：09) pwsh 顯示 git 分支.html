windows terminal pwsh 如何顯示 git branch 提示

在 Windows Terminal 的 PowerShell (pwsh) 中，可以利用 posh-git 模組來顯示 Git 分支提示。以下是簡單的設定步驟：

  1. 安裝 posh-git 模組
    在 PowerShell 中執行以下命令（建議使用目前使用者範圍）：

    powershell
    Install-Module posh-git -Scope CurrentUser

  2. 修改 PowerShell 組態檔
    編輯您的 PowerShell 組態檔（通常位於 ~\Documents\PowerShell\Microsoft.PowerShell_profile.ps1 或 $PROFILE 所指定的位置），並加入下列內容：

    powershell
    Import-Module posh-git

    如果您的檔案不存在，可以自行建立一個。

  3. 重新啟動 PowerShell
    儲存檔案後，重新開啟 Windows Terminal 的 pwsh 或執行：

    powershell
    . $PROFILE

    這樣您就能在進入 Git 資料夾時看到目前所在的分支提示。

此外，也可以考慮使用 oh-my-posh 模組，它提供更豐富的提示字元主題與自訂功能，但基本的 Git 分支提示功能在 posh-git 中已經足夠。
